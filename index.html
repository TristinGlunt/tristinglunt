Random Forests

In our graduate machine learning course, we implemented random forests using Python to classify intron-exon decision boundaries for DNA sequences. The random forest has now been made to mimic the training/predict pattern of scikit-learns training patterns. You could import the library the same way, and it should work on your dataset with almost identical accuracy as sklearns accuracy.

Naive Bayes and Logistic Regression

Again, in the UNM machine learning graduate course, we implemented Naive Bayes and Logistic Regression to classify news articles as one of twenty possible classifications. The NB and Log. Reg. implementation are based on Tom Mitchell's chapter on the two subjects. Again, the library has been implemnted to mimic the object oriented approach used by sklearn. The figure to right is a confusion matrix modeled by a heatmap showing the correlation of predicted and true classes.

Professional Gamer - AirCougar
